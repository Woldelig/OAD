Oppgave B - Side 2



Modellen har også ansvar for å returnere informasjon som dimensjonen til tabellen, datatypen til hver kolonne, kolonnetittel og om det er mulig å redigere verdien i en enkelt celle. Med disse ni metodene kan interfacet virke komplekst og tidkrevende å implementere. Java tilbyr DefaultTableModel som er en subklasse av AbstractTableModel hvor interfacet TableModel implementeres slik at det krever minimalt med innsats [1][2].

Ved å utvide klassen AbstractTableModel må man overstyre tre enkle metoder. Disse metodene gjør det mulig å returnere antall rader, antall kolonner og verdien til en spesifikk celle.

public int getRowCount();
public int getColumnCount();
public Object getValueAt(int row, int column);
